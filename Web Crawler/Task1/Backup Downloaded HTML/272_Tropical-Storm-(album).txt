<p><i><b>Tropical Storm</b></i> is <a href="/wiki/Reggae" title="Reggae">reggae</a>, <a href="/wiki/Dancehall" title="Dancehall">dancehall</a> artist <a href="/wiki/Beenie_Man" title="Beenie Man">Beenie Man</a>'s fifteenth studio album. It was released on August 20, 2002. The album is a mix of <a href="/wiki/Dancehall" title="Dancehall">dancehall</a> and <a href="/wiki/Reggae_fusion" title="Reggae fusion">reggae fusion</a>, with hit singles such as "<a href="/wiki/Feel_It_Boy" title="Feel It Boy">Feel It Boy</a>" featuring <a class="mw-redirect" href="/wiki/R%26B" title="R&amp;B">R&amp;B</a> singer <a href="/wiki/Janet_Jackson" title="Janet Jackson">Janet Jackson</a> and "<a href="/wiki/Bossman_(song)" title="Bossman (song)">Bossman</a>" featuring <a href="/wiki/Dancehall" title="Dancehall">dancehall</a> artists <a href="/wiki/Lady_Saw" title="Lady Saw">Lady Saw</a> &amp; <a href="/wiki/Sean_Paul" title="Sean Paul">Sean Paul</a>.</p>
