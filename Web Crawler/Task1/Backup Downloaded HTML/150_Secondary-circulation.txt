<p>A <b>secondary circulation</b> is a circulation induced in a rotating system. For example, the primary circulation of Earth's atmosphere is <a href="/wiki/Zonal_and_meridional" title="Zonal and meridional">zonal</a>. If however a parcel of air, that moves in a purely zonal direction, is accelerated or decelerated zonally, the <a href="/wiki/Coriolis_force" title="Coriolis force">Coriolis force</a> will add a <a class="mw-redirect" href="/wiki/Meridional" title="Meridional">meridional</a> component to its velocity. This meridional circulation is then the secondary circulation.</p>
