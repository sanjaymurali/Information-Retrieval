<p>The <b>meridian 130° east of <a class="mw-redirect" href="/wiki/Prime_Meridian" title="Prime Meridian">Greenwich</a></b> is a line of <a href="/wiki/Longitude" title="Longitude">longitude</a> that extends from the <a href="/wiki/North_Pole" title="North Pole">North Pole</a> across the <a href="/wiki/Arctic_Ocean" title="Arctic Ocean">Arctic Ocean</a>, <a href="/wiki/Asia" title="Asia">Asia</a>, <a href="/wiki/Australia" title="Australia">Australia</a>, the <a href="/wiki/Indian_Ocean" title="Indian Ocean">Indian Ocean</a>, the <a href="/wiki/Southern_Ocean" title="Southern Ocean">Southern Ocean</a>, and <a href="/wiki/Antarctica" title="Antarctica">Antarctica</a> to the <a href="/wiki/South_Pole" title="South Pole">South Pole</a>.</p>
<p>The 130th meridian east forms a <a href="/wiki/Great_circle" title="Great circle">great circle</a> with the <a href="/wiki/50th_meridian_west" title="50th meridian west">50th meridian west</a>.</p>
