<p>The <b>bibcode</b> (also known as the <b>refcode</b>) is a compact identifier used by several <a href="/wiki/Astronomy" title="Astronomy">astronomical</a> data systems to uniquely specify literature references.</p>

<h2><span class="mw-headline" id="Adoption">Adoption</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Bibcode&amp;action=edit&amp;section=1" title="Edit section: Adoption">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The Bibliographic Reference Code (refcode) was originally developed to be used in <a href="/wiki/SIMBAD" title="SIMBAD">SIMBAD</a> and the <a href="/wiki/NASA/IPAC_Extragalactic_Database" title="NASA/IPAC Extragalactic Database">NASA/IPAC Extragalactic Database</a> (NED), but it became a de facto standard and is now used more widely, for example, by the NASA <a href="/wiki/Astrophysics_Data_System" title="Astrophysics Data System">Astrophysics Data System</a> who coined and prefer the term "bibcode".<sup class="reference" id="cite_ref-a_1-0"><a href="#cite_note-a-1">[1]</a></sup><sup class="reference" id="cite_ref-b_2-0"><a href="#cite_note-b-2">[2]</a></sup></p>
<h2><span class="mw-headline" id="Format">Format</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/w/index.php?title=Bibcode&amp;action=edit&amp;section=2" title="Edit section: Format">edit</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>The code has a fixed length of 19 characters and has the form</p>
<center><tt>YYYYJJJJJVVVVMPPPPA</tt></center>
<p>where <tt>YYYY</tt> is the four-digit year of the reference and <tt>JJJJJ</tt> is a code indicating where the reference was published. In the case of a journal reference, <tt>VVVV</tt> is the volume number, <tt>M</tt> indicates the section of the journal where the reference was published (e.g., <tt>L</tt> for a letters section), <tt>PPPP</tt> gives the starting page number, and <tt>A</tt> is the first letter of the last name of the first author. Periods (<tt>.</tt>) are used to fill unused fields and to pad fields out to their fixed length if too short; padding is done on the right for the publication code and on the left for the volume number and page number.<sup class="reference" id="cite_ref-a_1-1"><a href="#cite_note-a-1">[1]</a></sup><sup class="reference" id="cite_ref-b_2-1"><a href="#cite_note-b-2">[2]</a></sup> Page numbers greater than 9999 are continued in the <tt>M</tt> column. The 6-digit article ID numbers (in lieu of page numbers) used by the Physical Review publications since the late 1990s are treated as follows: The first two digits of the article ID, corresponding to the issue number, are converted to a lower-case letter (01 = a etc.) and inserted into column <tt>M</tt>. The remaining four digits are used in the page field.<sup class="reference" id="cite_ref-b_2-2"><a href="#cite_note-b-2">[2]</a></sup></p>
